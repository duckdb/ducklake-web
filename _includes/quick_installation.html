Create your first DuckLake with DuckDB

DuckDB provides first-class support for DuckLake through its highly portable extension, running wherever DuckDB does.

  • DuckDB
  • PostgreSQL
  • SQLite
{% highlight sql %} INSTALL ducklake; ATTACH 'ducklake:mylake.ducklake' AS my_lake; ATTACH ducklake:postgres:dbname=... DATA_PATH ... METADATA_SCHEMA ... FROM my_lake.XX; {% endhighlight %}
{% highlight bash %} INSTALL ducklake; ATTACH 'ducklake:mylake.ducklake' AS my_lake; ATTACH ducklake:postgres:dbname=... DATA_PATH ... METADATA_SCHEMA ... FROM my_lake.XX; {% endhighlight %}
